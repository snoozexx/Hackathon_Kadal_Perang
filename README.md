# Hackathon_Kadal_Perang
imphen hackathon team kadal perang, pak edy army

( DATA DUMMY )
1. Nyalakan ESP8266
2. Di Website nya pilih id kendaraan yg ingin di deteksi
3. Kemudian, AI nya akan mendeteksi kerusakan dan memberikan rekomendasi perbaikan nya

( Real Pakai Sensor )
1. Nyalakan ESP
2. Hubungkan dengan bluetooth atau tancapkan sensor nya pada ECU kendaraan
3. AI nya akan mendeteksi dan mengambil data nya
4. Jika yg di ambil data kendaraan nya itu baru pertama kali alias belum pernah maka otomatis akan menambahkan data kendaraan baru,
5. Sedangkan jika kendaraan tsb sudah ada atau sudah pernah di deteksi sebelumnya maka akan mengambil data kendaraan yg sudah ada
6. Di Website nya pilih kendaraan yg ingin di deteksi
7. Kemudian AI nya akan mendeteksi dan memberikan rekomendasi perbaikan pada kendaraan nya
